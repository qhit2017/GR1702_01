1.面向对象的三大特征是什么，分别有什么好处？
面向对象是一种程序设计的思想
继承
多态
抽象

好处:封装:对于简单的数据类型,有它自己的封装类;对于引用类型，类就是一个封装体的     一个表现。
继承:子类继承父类，它具有可扩展性，便于程序代码的重用性。继承可以实现多态:通过     重写,通过对象上传可以实现。
多态:通过重写和重载实现。

2.面向对象的三大特征，分别是如何实现的？
 1.封装:也就是把客观事物封装成抽象的类，并且类可以把自己的数据和方法只让可信的类或者对象操作，对不可信的进行信息隐藏。

 2.继承实现方式有三类：实现继承、接口继承和可视继承。

   实现继承:是指使用基类的属性和方法而无需额外编码的能力。

   接口继承:是指仅使用属性和方法的名称、但是子类必须提供实现的能力。
  
   可视继承:是指子窗体（类）使用基窗体（类）的外观和实现代码的能力。


 3.实现多态有二种方式:覆盖，重载。
