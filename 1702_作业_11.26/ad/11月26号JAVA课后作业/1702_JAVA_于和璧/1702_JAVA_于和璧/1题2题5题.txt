1、 面向对象的三大特征是什么，分别有什么好处？

三大特征：封装，继承，多态
好处：
	封装：限制在类的外部直接修改类的属性，保证了数据的安全性和有效性。
	继承：简化了代码，实现了代码的重用性。　
	多态：提高了代码的维护性(继承保证)，提高了代码的扩展性(由多态保证)。
                 


2、面向对象的三大特征，分别是如何实现的？
	封装：（1）修改属性的可见性，限制访问。    （2）设置属性的读取方法。    （3）在读取属性的方法中，添加对属性读取的限制。
	继承：在子类类名后输入extends 后跟父类类名，则实现对父类的继承。
	多态：多态在JAVA中可以分别通过方法的重写与重载来实现。
