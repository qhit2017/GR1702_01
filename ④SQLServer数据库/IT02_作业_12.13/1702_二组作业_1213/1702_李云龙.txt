4、安装sqlserver的步骤是什么？
     双击SQL Server 2008的.exe安装文件进入SQL S erver 安装中心
     点击界面左侧的安装，然后点击右侧的全新SQL Server 独立安装或向现有安装添加功能，进入SQL S erver 2008 安装程序界面，首先是安装程序支持规则，操作完成之后，点击确定
        进入许可条款，勾选上我接受许可条款，点击下一步
       进入安装程序支持文件，点击安装按钮，开始安装支持文件
       安装完成之后，又进入安装程序支持规则，点击显示详细的规则列表，点击下一步
       进入功能选择，这里选择全选，然后点击下一步
       进入实例配置，选择命名实例，点击下一步
       进入磁盘空间要求，显示磁盘的使用情况，可以直接点击下一步
       进入服务器配置，单击对说所有SQL Server 服务使用相同的账户，选择NT AUTHORITY\SYSTEM,然后点击下一步
       进入数据库引攀配置，单击添加当前用户指定SQL Server管理员，这样管理员就是系统管理员，设置好之后，单击下一步
       进入Reporting Services配置，全选，然后点击下一步
       进入错误和使用情况报告，然后点击下一步
       进入安装规则，点击下一步
       进入安装进度SQL Server 2008开始安装，安装完成后点击下一步，进入完成界面，然后点击完成
       
5、数据库中表、元组、行、列、属性分别是什么意思？
       表：数据库中的表与我们日常生活中使用的表格类似，它也是由行
       和列组成的
       
       元组：表中的每一行就是元组
       行：行包括了若干列。一行数据成为一个或一条记录，它表达有一定
       意义的信息组合
       列：列由同类的信息组成，每列又称为一个字段、属性的域，每列的
       标题称为字段名或属性名
       
 属性：列就是属性
